[[udp-input]]
== UDP input

++++
<titleabbrev>UDP</titleabbrev>
++++

Use the `udp` input to read events over UDP.

Example configuration:

[source,yaml]
----
inputs:
  - type: udp
    max_message_size: 10KiB
    host: "localhost:8080"
----


The `udp` input supports the following settings, grouped by category.
Many of these settings have sensible defaults that allow you to run {agent}
with minimal configuration.

<<input-udp-udp-settings>>::
Settings for reading over UDP.

Settings used to parse, filter, and transform data.
