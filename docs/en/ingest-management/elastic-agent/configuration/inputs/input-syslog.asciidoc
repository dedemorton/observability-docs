[[syslog-input]]
= Syslog input

Use the `syslog` input to read events over TCP, UDP, or a Unix stream socket.
This input will parse BSD (rfc3164) events and some variants.

Example configurations:

[source,yaml]
----
inputs:
  - type: syslog
    protocol.udp:
      host: "localhost:9000"
----

[source,yaml]
----
inputs:
  - type: syslog
    protocol.tcp:
      host: "localhost:9000"
----

[source,yaml]
----
inputs:
  - type: syslog
    protocol.unix:
      path: "/path/to/syslog.sock"
----
